Introduction

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browsers receive HTML documents from a webserver or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document. Web pages can be created and modified by using professional HTML editors. However, for learning HTML text editors like Notepad (PC) or TextEdit (Mac) will do good.

Nesting

HTML documents consist of nested HTML elements. What does that mean ?
The easiest way to understand nesting in HTML is to imagine tags as boxes that hold your content. This can include text, images, hyperlink, etc. In most cases, you have to place boxes in other boxes. The element, which is inside other box, is the nested element and you have to close the inner tag before and outside one.

Headings and Paragraphs

HTML defines six headings tags. The heading implies all the font changes, paragraphs etc. before and after. The heading tags are h1, h2, h3, h4, h5, h6. H1 is the biggest and h6 is the smallest element.
The p element represents a paragraph of text. They are usually represented as visual media as block of text. In order to go to a new line you should open a new p tag or to use br(break line).

Formatting

This is maybe the largest section, when it comes to number of elements. They range from extremely useful and recently used to little-known. All formatting elements are inline and they can’t be wrapped around any block-level elements. There exceptions like del and ins, because they have special properties.

Colors

HTML colors are combination of the colors red, green and blue. They can be displayed in hexadecimal(#FF0000 – this color is red), rgb(0,255,0 – color green) or just typing the well-known colors.

Images

In order to put image on a webpage we use the “img” tag. This element is empty and requires at least one attribute to work – src(contain path point to the source of the image), alt (description of the image) and title(which adds tooltip to the image).
