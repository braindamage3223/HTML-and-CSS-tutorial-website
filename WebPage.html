Why HTML and CSS ?

Understand the Web
Learning the basics of HTML and CSS will help you to understand the web better than before. You will be able to create web pages on your own using and style them according to your taste.
Many new companies are looking for web developers to create their webpages. Learning CSS and HTML will provide some great opportunities for you, and if you add, some JavaScript to your learnings will make your task even easier.

Become a professional Web developer
Nowadays, most developers use free templates to create their webpages. You can do the same thing but your website will look like every other one on the Web.. If you learn HTML and CSS you will be able to create your own website and you won’t have to pay anyone to do it for you.

Start a Web Career
If you want to be a professional web developer learning HTML and CSS is a must! You can search in Google about the importance of learning HTML and CSS if you are planning to start a web career.

Learn Another Languages easier
Learning HTML and CSS will make your life easier in the future, if you are planning to be a professional developer. For example, it will be a lot easier to learn JavaScript, PHP etc. and of course, after that the future is in your hands.

Make profit
You can start making money by learning HTML and CSS. You can search in Google and you will find that there are dozens of freelance jobs.It will be easier of course, if you learn some JS, PHP and maybe some photoshopping.

Is it difficult ?

You can learn HTML and CSS on your own; you don’t need a degree or something. HTML and CSS are the easiest programming languages to learn. Java or C++ are difficult because of how they are explained. However, you can start learning HTML and CSS in small steps.
Your first web page probably won’t be very pretty, but with learning and practicing you can be a professional one day.
